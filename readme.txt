If you want to run this code locally, paste the following command in your terminal:
    npm i
    
If it's not working, try this instead:
    npm install express axios

I use nodemon to run my code locally. If you want to use too, install this running this code in your terminal:
    npm install nodemon

After this, follow the instrucions:
    in your terminal, type: nodemon index.js
    in your favourite browser, paste the following link:
        http://localhost:3001/
